## Overview
This project demonstrates how agentic AI systems can deliver personalized,
adaptive learning across subjects using coordination, memory, and evaluation.

## Architecture
- Coordinator Agent
- Math Tutor Agent
- Science Tutor Agent
- Writing Tutor Agent
- Memory Bank (long-term learning context)
- Evaluation Agent (performance metrics)

## Key Features
- Multi-agent collaboration
- Personalized learning paths
- Long-term memory for continuity
- Adaptive difficulty adjustment
- Evaluation and observability

## Tech Stack
- Python
- FastAPI
- asyncio
- Docker

## Use Case
Designed to make high-quality, personalized education accessible and scalable,
especially for learners with different strengths and learning styles.
